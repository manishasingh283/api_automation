## Configure

### Installation
install cypress 9.7.0
Extract file
npm i




## Run
Go to terminal - [npm run cypress-headed]
### Default
Cypress can be run in two modes -
1. `Headed` (browser is displayed)
    ```
    npm run cypress-headed
    ```
2. `Headless` (command line only)
    ```
    npm run cypress
    ````
